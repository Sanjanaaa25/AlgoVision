# AlgoVision

AI-powered DSA Problem Analyzer and Hint Generator

## Features

* Identifies the optimal algorithm or pattern for a DSA problem
* Estimates problem difficulty
* Provides guided hints without revealing the complete solution
* Explains the core idea behind the approach
* Helps students improve problem-solving skills

## Tech Stack

### Frontend

* React
* TypeScript

### Backend

* FastAPI
* Python

### AI Integration

* OpenAI API

## How It Works

1. User enters a DSA problem statement.
2. AI analyzes the problem.
3. Pattern and algorithm are identified.
4. Hints are generated.
5. Complexity and explanation are returned.
